# Non Nemo Layout

The option(⌥), shift(⇧), control(⌃) and command(⌘) keys are assigned to home position keys on each side.

## Thumb keys

| Tap         |  Long tap        |
| ---         |  ---             |
| ESC         |  Navigation layer|
| Tab(⇥)      |  Function layer  |
| Space(␣)    |  Code layer      |
| Enter(⏎)    |  Code layer      |
| Backspace(⌫)|  Function layer  |
| Delete(⌦)   |  Media layer     |
